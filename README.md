# form-rag-js

**One AI form. Any small business. Powered by Grok.**

## Install

```bash
npm install form-rag-js
```

## Use

```jsx
import { FormRAG } from 'form-rag-js';
import { UniversalSMBTemplate } from 'form-rag-js/templates/universal-smb.js';

<FormRAG template={UniversalSMBTemplate} apiKey="gsk_..." />
```
